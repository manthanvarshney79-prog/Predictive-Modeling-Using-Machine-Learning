# Predictive Modeling Using Machine Learning

## Objective

Build a machine learning model to predict passenger survival using the Titanic dataset.

## Dataset

Titanic Dataset

## Algorithm Used

Logistic Regression

## Libraries Used

* Pandas
* Scikit-learn

## Steps Performed

1. Loaded the dataset.
2. Handled missing values.
3. Converted categorical data into numeric form.
4. Split data into training and testing sets.
5. Trained a Logistic Regression model.
6. Evaluated model accuracy.

## Result

The model successfully predicted passenger survival and achieved a reasonable accuracy score.

## Conclusion

Machine learning techniques can be used to make predictions from historical data and evaluate performance using accuracy metrics.
